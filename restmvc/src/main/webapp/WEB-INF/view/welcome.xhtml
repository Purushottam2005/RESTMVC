This JEE6 Web application has the following features.

  • JAX-RS with Jersey is used as the Controller
  • JPA Entity POJOs with JSON annotations are used as the Model
  • Stateless JSF/Facelets templates are used as the View
  • The Jersey View API is used as the glue between the Controller and the View
  • JAX-RS extensions with the Jackson JSON API are used to consume and produce between JSON and the Model
  • Stateless Session EJB services with EJB lite are integrated into the JAX-RS views and resources
  • Contexts and Dependency Injection with managed beans are utilized throughout
  • Semantic HTML5 mark-up
  • CSS3 Responsive Web Design
  • Modernizr and jQuery JavaScript libraries
  • Production mode features for browser performance
